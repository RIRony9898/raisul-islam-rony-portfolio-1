## üöÄ Getting Started

### Prerequisites

- Node.js (version 14 or higher)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/raisul-islam-rony-portfolio-1.git
   cd raisul-islam-rony-portfolio-1
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the development server:

   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173` to view the portfolio.
